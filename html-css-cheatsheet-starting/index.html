CSS Selectors Cheat Sheet

Common Selectors

Type Selector table { }
Just write the element name and the CSS selector will target all elements on the page.
Class Selector.contrainer {}
Classes are selected with a dot(.) at the beginning of them.
ID Selector (Not recommended) #content { }
ID's are selected with a hashtag at the beginning. I don't personally use ID's a lot to target elements.

Descendant and Child Selectors

Descendant Selector .container p { }
Selects all the descendants of a parent selector In this example, all the p elements under the .container class are selected, even though the p elements are nested under other elements. Descendant selectors select all child elements of its parent.
Child Selector .container > p { }
Child selectors only select the direct child of its parent. Compared to the example given above, only the direct child p of .container gets targeted.

Pseudo Selectors

First Child Pseudo Selector div p:first-child{ }
Targets the first child of a patent. If the first child is not the same as the targeted element, then nothing gets selected. in this example the first child p of the div element gets selected. However, if the first child is the div is eg. an h1 and not a p then nothing gets selected.
Last Child Pseudo Selector div p:last-child{ }
Targets the last child of a patent. If the last child is not the same as the targeted element, then nothing gets selected. in this example the last child p of the div element gets selected. However, if the last child is the div is eg. an footer and not a p then nothing gets selected. (this doesn't work on body element so the parent should not be a body element but a section or div )
Only Child Pseudo Selector strong:only-child{ }
Targets the first parent which has a single child of the targeted element of a patent. In this example. p has a single strong element. So this strong element is targeted.
(n)th Child Pseudo Selector p:nth-child(2){ }
Targets the n'th child of the parent. If they are siblings targets the nth child which matches the targeted element.
(n)th Last Child Pseudo Selector p:nth-last-child(2){ }
Targets the last n'th child of the parent. If they are siblings targets the nth child which matches the targeted element.
